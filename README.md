# 世界の理を取り戻す旅 ─ 数学RPG

> 中学1年生向け 2Dトップダウン型 数学教育RPG（デモ版 / Ch1）

---

## 🎮 ゲーム概要

「数」の概念が世界から奪われた！
プレイヤーはダンジョンを探索し、**数学の問題に答えて敵を倒し**、世界の理の「欠片」を集めてボスに挑む。

### 現在プレイできる範囲（デモ版）
- **第1章「氷と炎の谷」**（正負の数：加減乗除）
- ダンジョン4階層 + ボス戦（氷炎の魔人）
- 章クリア後にエンディングシーン

## 🗺 ゲームの流れ

1. **Googleアカウント** でログイン
2. プロローグを読み、冒険へ出発
3. **村** でショップに立ち寄り、装備を整える
4. **ダンジョン入口（青タイル）** からダンジョンへ
5. 徘徊する敵シンボルに接触 → バトル開始
6. **数学の問題に答えて** 攻撃 / ポーションで回復
7. 欠片を10個集めて **封印の間（4F）** へ
8. ボスを倒してクリア！

---

## ⚔ バトルシステム

- 正解 → 敵にダメージ（連続正解でコンボボーナス！）
- 不正解 → ミス（ダメージなし、敵は必ず反撃）
- **3連続正解** → ダメージ×1.5倍
- **5連続正解** → ダメージ×2.0倍
